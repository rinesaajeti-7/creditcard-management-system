# Credit Card Management System

Një sistem për menaxhimin e kartave të kreditit, ndërtuar me ASP.NET Core.

## Features
- Regjistrim dhe login i përdoruesve
- Shtimi, editimi dhe fshirja e kartave të kreditit
- ...

## Si ta nisësh projektin
1. Clone repo-n
2. Hap me Visual Studio ose VS Code
3. dotnet add package BCrypt.Net-Next
4. dotnet add package Microsoft.EntityFrameworkCore
5. dotnet add package Npgsql.EntityFrameworkCore.PostgreSQL
6. dotnet add package Microsoft.EntityFrameworkCore.Design
7. dotnet add package Microsoft.AspNetCore.Authentication.Cookies
8. dotnet add package Microsoft.AspNetCore.Authentication.JwtBearer
9. npm init -y
10. npm install bootstrap
11. dotnet restore
12. dotnet ef migrations add InitialCreate
13. dotnet ef database update
14. dotnet build
15. dotnet run
